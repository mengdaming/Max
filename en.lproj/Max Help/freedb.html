FreeDB is an online service akin to CDDB that is a user-driven effort to provide information to automatically tag CDs for ripping.

Use

You can have Max automatically query FreeDB when a CD is inserted by selecting Automatically acquire artist/track names under General in Max's preferences.

Otherwise, if you would like to do it manually, when the Compact Disc window is open, you can click Query FreeDB in the toolbar.

Multiple matches

Occasionally, FreeDB will return multiple matches for your CD, some of which will contain minor differences from each other, or, rarely, information from entirely different CDs. Choose the closest match to yours. You can always change the information before you rip the CD.

Configuration

Configuration settings for FreeDB can be found under FreeDB in Max's preferences.

Servers

The preferred server is what Max will query to obtain your CDs information, and what Max will use for you to submit information. Max requires a server's port number and the protocol used to access its services. Port 80 is used for the HTTP protocol; port 8880 is used for the CDDBP protocol. If you choose to use the HTTP protocol the you must append /~cddb/cddb.cgi to the preferred server name.

You can refresh the list of available mirrors to select a mirror closer to you for faster access or if your preferred server is inoperative.

Submitting CD information

If FreeDB doesn't contain information for your CD or if you find incorrect information, then you can add or correct information in Max and submit it to FreeDB by selecting Submit to FreeDB from the Compact Disc menu. FreeDB requires an e-mail address (not neccesarily a working one) in the form of username@host.
