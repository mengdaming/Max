Max uses cdparanoia to rip CDs. cdparanoia is a meticulous ripping engine that can compensate for scratched CDs and poor CD drives to produce flawless rips.

Configuration settings for cdparanoia can be found under Ripper in Max's preferences.

Logging

cdparanoia's log will keep a detailed record of its ripping status. If Logging is enabled, the log window can be found under the Window menu.

Error Correction

Error correction is what enables cdparanoia to rip pristine audio from scratched CDs and deficient CD drives. Enabling error correction is slower than ripping without it because cdparanoia verifies each file and attempts to correct any errors it finds.

Overlap checking

Overlap checking is useful for old or defective CD drives that don't read the precise data block that cdparanoia requests. When cdparanoia collects those blocks into a sequence, then the resulting file will be defective. cdparanoia can use overlap checking to circumvent the CD drive by checking the blocks itself and properly aligning them in their proper sequence.

CD Skip Behavior

If an error is found while ripping, cdparanoia will attempt to correct it.

If Never allow skipping is selected, then cdparanoia will never give up trying to correct the error. If the error is too substantial, then cdparanoia will effectively stall.

If Never allow skipping is deselected, then you can assign the amount of times you would like cdparanoia to attempt to correct the error before giving up and moving on.
